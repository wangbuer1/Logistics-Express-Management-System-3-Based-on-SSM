# 物流/快递管理系统3



## 1、项目介绍

物流/快递管理系统3只有一个管理员角色，包含公告管理、仓库管理、供应商管理、物品管理、入库/出库管理/顾客信息查询等功能，适合简单的技术学习


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.5-5.7、8.x版本都可） 
- maven项目：否
- 硬件环境：Windows
